// Portfólio de Projetos

PC-Builder - Montador de PCs

Ver Online Ver Código

Desenvolvimento de uma aplicação web full-stack, concebida para simular a experiência de montar um computador personalizado. A plataforma permite que os utilizadores selecionem componentes individuais, valida a sua compatibilidade em tempo real, calcula o custo total e permite guardar e partilhar as configurações.

Principais Funcionalidades:

Arquitetura e Back-end (Laravel):

  • Estruturação de uma API RESTful para gerir todas as operações de CRUD (Criar, Ler, Atualizar, Apagar) para componentes, utilizadores e configurações guardadas.
  • Utilização do Eloquent ORM para modelar a base de dados e gerir os relacionamentos complexos entre as tabelas de componentes (CPU, Motherboard, RAM, etc.).
  • Implementação da lógica de negócio para o sistema de validação de compatibilidade, que verifica as regras entre as peças selecionadas (ex: compatibilidade de socket do CPU com a motherboard).

Front-end e Experiência do Utilizador (Vue.js & Inertia.js):

  • Criação de uma interface reativa no estilo Single-Page Application (SPA), proporcionando uma navegação fluida e sem recarregamentos de página.
  • Desenvolvimento de componentes Vue reutilizáveis (como seletores de peças, sumário da montagem e modais de informação) para manter o código organizado e escalável.
  • Gestão do estado da aplicação no lado do cliente para refletir as escolhas do utilizador em tempo real.

Design e Visualização de Dados (TailwindCSS & Chart.js):

  • Construção de uma interface de utilizador moderna e totalmente responsiva com o framework utilitário TailwindCSS.
  • Integração da biblioteca Chart.js para renderizar gráficos dinâmicos, como o histórico de preços de um determinado componente, consumindo dados da API.
